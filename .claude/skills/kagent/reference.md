# Kagent Reference

## Overview
Kagent is an advanced AI agent for Kubernetes that provides deep insights, optimization recommendations, and automated troubleshooting.

## Key Capabilities
- **Analysis**: Deep dive into cluster performance and resource usage.
- **Optimization**: Suggest resource limits, requests, and scaling policies.
- **Troubleshooting**: Root cause analysis for failures.
- **Reporting**: Generate summaries of cluster health.

## Command Syntax
```bash
kagent "natural language request"
```

## Comparisons
- **kubectl-ai**: Focuses on generating/applying manifests (Action oriented).
- **kagent**: Focuses on analysis, optimization, and reasoning (Insight oriented).
